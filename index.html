Single Page Web Applications (SPAs)

Single Page Web Applications run in a browser without reloading from the server during execution. SPAs are thus rich clients running practically anywhere without installation. Examples for SPAs would be Gmail™, Twitter™, Facebook™, and Microsoft Office 365™.

The technologies I like and use to write SPAs include Java, Spring, Hibernate, GWT, JBoss Errai, Vaadin, JavaScript, jQuery, Bootstrap, AnguluarJS, MongoDB and TypeScript.
